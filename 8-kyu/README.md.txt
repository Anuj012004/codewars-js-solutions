# Codewars JavaScript Solutions

This repository contains my JavaScript solutions to Codewars challenges, organized by difficulty level.

## Difficulty Levels
- `8-kyu`: Beginner-friendly problems.
- `7-kyu`: Intermediate problems.
- `6-kyu`: More advanced problems.

## About Codewars
[Codewars](https://www.codewars.com/) is a platform where developers solve coding challenges to improve their skills.
